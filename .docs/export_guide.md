# エクスポート機能ガイド

## 対応フォーマット

- **CSV**: 標準形式
- **Excel**: カスタマイズ可能
- **PDF**: 月次レポート

---

## CSV エクスポート

### 基本使用

1. `/transactions` ページを開く
2. フィルタ条件を設定（任意）
3. 「エクスポート」→「CSV」を選択
4. ダウンロード開始

### 出力カラム

- 取引日
- 内容
- 取引先
- 金額
- カテゴリ
- 事業フラグ
- 按分率

---

## Excel エクスポート

### テンプレート機能

`/templates` ページでテンプレート作成:
- カラム選択
- 並び順設定
- フィルタ保存

### 使用方法

1. `/transactions` ページ
2. 「エクスポート」→「Excel」
3. テンプレート選択
4. ダウンロード

### フォーマット

- ヘッダー行: 青背景・白文字
- 金額: カンマ区切り・¥記号付き
- 按分率: パーセント表示

---

## PDF レポート

### 月次レポート

1. `/reports` ページ
2. 年月を選択
3. 「PDF生成」クリック

### 含まれる情報

- サマリー統計
  - 総支出
  - 事業支出
  - 個人支出
  - 取引件数

- カテゴリ別内訳
  - 各カテゴリの金額
  - 割合

---

## エクスポート履歴

`/exports` ページで確認:
- 実行日時
- フォーマット
- フィルタ条件
- ファイルサイズ

## カスタムエクスポート

### API使用

```typescript
const response = await fetch('/api/exports/transactions', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    format: 'csv',
    filters: {
      start_date: '2024-01-01',
      end_date: '2024-01-31',
      is_business: true
    }
  })
});

const blob = await response.blob();
```

## トラブルシューティング

### 問題: エクスポートに時間がかかる

**原因**: データ量が多い

**対処法**: フィルタでデータ範囲を絞る

### 問題: Excelファイルが開けない

**原因**: 形式エラー

**対処法**: OpenOfficeまたはGoogle Spreadsheetsで開く

---

## ベストプラクティス

1. **定期エクスポート**: 月末に1回実行
2. **バックアップ**: ローカルに保存
3. **テンプレート活用**: よく使う設定を保存
4. **フィルタ活用**: 必要なデータのみエクスポート
