# Phase 3: 証憑管理強化＋データ出力（必要になったら）

## 1. 目的
- 証憑（レシート・領収書）の管理を強化する。
- 必要に応じて外部ツール（Excel、会計ソフト）にデータを出力できるようにする。

## 2. スコープ

### 2.1 OCR連携
- receiptsへの画像アップロード強化。
- OCRによる自動テキスト抽出（ocr_text, ocr_confidence）。
- 金額・日付の自動抽出と取引への紐付け候補表示。

### 2.2 データエクスポート機能
- 汎用CSV出力（期間指定、カラム選択可能）。
- Excel形式出力。
- 出力テンプレートの保存・再利用。

### 2.3 メール領収書取り込み
- Gmailから領収書PDFを自動検出。
- 添付ファイルをSupabase Storageに保存し、receiptsテーブルに自動登録。

## 3. 受け入れ基準

### 3.1 OCR機能
- [ ] 画像/PDFアップロード時にOCRが自動実行され、結果が保存される。
- [ ] OCRから抽出された金額・日付に基づき、取引への紐付け候補が提案される。

### 3.2 エクスポート機能
- [ ] 期間・列を指定して汎用CSV/Excel形式で出力できる。
- [ ] 出力テンプレートを保存・再利用できる。
- [ ] 未判定取引がある場合に警告が表示される。

### 3.3 メール領収書
- [ ] Gmailから領収書メールを検索し、添付PDFを自動ダウンロード・保存できる。
- [ ] メール本文から金額・日付を抽出できる。
