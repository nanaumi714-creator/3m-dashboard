# 🎯 成功の定義

## Phase 1 成功基準
- [ ] 1ヶ月分のカード明細（30-50件）を取り込み、経費判定ができる
- [ ] 未判定件数をゼロにできる
- [ ] 月次レポートで支出傾向を把握できる
- [ ] ローカル環境でストレスなく動作する（3ヶ月分で実用速度）
- [ ] 証憑を取引に紐付けて参照できる
- [ ] **課金ゼロで運用できている**

## Phase 2 成功基準
- [ ] 表記ゆれが吸収され、同一店舗が統一的に扱える
- [ ] 判定作業時間が50%削減される（過去判定の再利用により）
- [ ] 複数の支払い手段（カード・口座・d払い・現金）を一元管理できる
- [ ] 証憑アップロードがWeb UIから快適に行える
- [ ] 重複候補を効率的に処理できる

## Phase 3 成功基準
- [ ] レシートをアップロードすると金額・日付が自動抽出される
- [ ] 必要に応じてExcelやCSVで出力できる
- [ ] Gmailから領収書を自動取り込みできる
- [ ] 高度な検索・分析で支出傾向を深掘りできる
- [ ] OCRコストが許容範囲内に収まっている

## Phase 4 成功基準
- [ ] 外出先からスマホでデータ確認できる
- [ ] レシート撮影→即アップロードができる
- [ ] 安全にクラウド上でデータが管理されている
- [ ] ローカル環境からスムーズに移行できた
- [ ] モバイルUIがストレスなく使える
- [ ] クラウドコストが許容範囲内に収まっている
